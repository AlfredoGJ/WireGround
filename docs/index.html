WireGround

A web application for circuit diagrams drawing

Objetivo

Realizar un aplicacion web que permita modificar, eliminar y guardar informacion utilizando para ello los conocimientos adquiridos durante el curso y que ayude a los estudiantes de computación e informática a facilitar el aprendizaje de circuitos basica.

Descripcion

Este proyecto constará en la elaboración de una aplicacion web dedicada a la creación de diagramas de circuitos logicos y circuitos integrados basicos, la cual será una herramienta muy útil para personas que estén buscando cada uno de estos circuitos y la manera de conectarlos, en la aplicación los usuarios podrán crear, ver, modificar y compartir los diagramas con otros usuarios.
